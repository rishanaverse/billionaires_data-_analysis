# billionaires data analysis🌏
An Excel-based data analysis project exploring the global billionaires dataset. This project includes data cleaning, analysis, and visualization to uncover insights about billionaire rankings, net worth, countries, industries, and wealth distribution using Excel tools such as Pivot Tables, formulas, and charts.


⸻

📌 Project Overview

Welcome to my Billionaires Data Analysis Project 📊

This project explores a global billionaires dataset using Microsoft Excel to uncover meaningful insights about wealth distribution, billionaire rankings, countries, and industries.

The goal of this project was to transform raw data into clear insights through data cleaning, analysis, and visualization techniques.

⸻

🎯 Project Goals

✔ Clean and organize raw billionaire data
✔ Analyze global wealth patterns
✔ Identify top billionaires and industries
✔ Explore country-wise billionaire distribution
✔ Create meaningful Excel-based visualizations
